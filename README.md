# 🔐 Cyber Threat Detection System (UIRTDS)

## 📌 Overview
A real-time malware and anomaly detection system that monitors file system activities and detects suspicious behavior using machine learning and rule-based analysis.

## 🚀 Features
- Real-time file monitoring (create, modify, delete)
- Anomaly detection using Isolation Forest
- Risk scoring system
- Flask-based dashboard for visualization
- System metrics tracking (CPU, memory)

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Watchdog
- Psutil

## ⚙️ How It Works
- Monitors file activity
- Extracts features
- Detects anomalies
- Assigns risk score
- Displays results in dashboard
